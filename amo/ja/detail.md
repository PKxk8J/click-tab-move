# Summary

増えすぎたタブ・グループを別のウィンドウやグループに簡単に移動できます。
クリックしたタブや全てのタブだけでなく、右側のタブやタブバーで選択済みのタブ、別ウィンドウで選択したタブ等、移動させるタブを切り替えられます。

# Description

ClickTabMove は、Firefox の右クリックメニューからタブやタブグループを移動するための拡張機能です。
タブをまとめてグループ化したいとき、複数ウィンドウをまとめたいとき、選択済みタブをまとめて移したいとき、タブグループを崩さず整理したいときにも使えます。

以下の移動対象を選べます。

- クリックしたタブ・グループ
- クリックしたタブ・グループより右側または左側のタブ・グループ
- クリックしたタブ・グループと、その右側または左側にある全てのタブ・グループ
- ウィンドウ内の全てのタブ・グループ
- タブバーで選択済みのタブ
- 選択ウィンドウでチェックしたタブ・グループ

さらに、移動対象ごとに、移動先を選べます。

- 新しいウィンドウ
- 別の既存ウィンドウ
- 新しいタブグループ
- 別の既存タブグループ

全てのタブを別ウィンドウへ移動すると、2つのウィンドウを統合できます。
タブグループを別ウィンドウへ移す場合は、グループのまま移動します。
タブを既存グループへ移す場合は、そのグループへ統合します。

グループそのものを選ぶ場合と、グループ内の全タブを選ぶ場合は別の移動対象として扱います。
選択ウィンドウでは、グループのチェックボックスで「グループを移動」「グループ内の全タブを移動」「どちらも移動しない」を切り替えられます。
タブバーで選択済みのタブは、グループ内の全タブが選ばれていてもグループではなく個別タブとして移動します。
分割ビューは1つのタブとして扱います。

ピン留めされたタブをウィンドウへ移す場合は、ピン留めされたタブのまま移動します。
ピン留めされたタブをグループへ移す場合は、毎回確認する、ピン留めされていないタブだけを移動する、ピン留めを外して移動する、から動作を選べます。

通知を有効にすると、処理の進行状況と完了結果を表示できます。

## プライバシー

ClickTabMove は、タブ、タブグループ、ピン留めされたタブ、分割ビューを移動するためにタブへのアクセス権限を使用します。
閲覧データを収集または送信することはありません。
