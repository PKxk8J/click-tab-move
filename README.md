# click-tab-move

タブ右クリックからタブを別のウィンドウへ移動させる Firefox 専用アドオン。

https://addons.mozilla.org/addon/clicktabmove/

## 機能

- このタブまたはこのグループを移動
- 右側・左側のタブやグループをまとめて移動
- グループ内の右側・左側のタブを移動
- すべてのタブ・グループを別ウィンドウへ移動
- ポップアップで選択したタブ・グループを移動
- タブやグループを新しいグループまたは既存グループへ移動
- 固定タブは固定タブのまま移動
- 設定で右クリックメニューの表示場所と移動対象を選択
- 通知は設定で有効にした場合のみ使用

すべてのタブを別ウィンドウへ移動すると、2 つのウィンドウを統合できます。

グループをウィンドウへ移す場合はグループのまま移動します。
グループを別のグループへ移す場合は移動元グループを解体して移動先グループへ統合します。
分割ビューは 1 つのタブとして扱います。
固定タブとすべてのタブ・グループはグループへ移動できません。

## 動作要件

- Firefox 142 以降
- Node.js 現行 LTS

## 開発

```sh
npm install
npm run lint
npm run test
npm run build
```

アドオンのバージョンは `extension/manifest.json` で管理します。
`npm run build` は `web-ext-artifacts/clicktabmove-<version>.zip` を作成します。

`npm run run` は、この拡張機能を一時的に読み込んだ Firefox を起動します。
拡張機能のソースは `extension/` にあります。

## プライバシー

この拡張機能はユーザーデータを収集または送信しません。
